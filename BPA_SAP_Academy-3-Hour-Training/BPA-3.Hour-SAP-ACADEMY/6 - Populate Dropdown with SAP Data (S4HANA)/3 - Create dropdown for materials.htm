Create dropdown for materials

  1. On your Create Sales Order page, go back to UI canvas by clicking View.

    You can close the logic pane if it interferes with your work.

  2. Drag a dropdown field next to the input box for the Material.

    It should look like this.

  3. Click on the dropdown box. Take note of the Label text, Option list and Selected value properties because you are about to change them.

  4. Set the Label text property to an empty string (i.e. delete Dropdown label value).

  5. For the Option list property, click the list of values icon, and set the binding to a formula, and set the formula to the following:

     
    MAP(data.A_ProductDescription1, {label: item.Product+ " (" + item.ProductDescription +") " ,value: item.Product})

  6. The Dropdown field properties should now look like:

  7. You will have to bind the dropdown field to the material page variable (as we did for the input field for material).

    For the Selected value field, set the binding to: Data and variables > Page variable > material.

    Click Save

    The Dropdown field properties should now look like:

  8. Click Save (upper right).
