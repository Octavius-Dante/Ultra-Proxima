Create an app project

Note that there are 2 different approaches for this tutorial step.

You can do the first approach, Copy Template Project, which will bypass the details of adding and stylizing components, and instead teach you how to copy an existing SAP Build Apps template project which has some of the UI components for this tutorial already built. This way is much faster.

You can instead do the second approach, Basic steps, which will teach you about UI components, stylizing them, and organizing them on the page.

In the interest of time, you will follow the Copy Template Project approach. However, the Basic steps approach is available as an optional exercise (see section [OPTIONAL] Create UI – Basic Steps).

Click Lobby to open the entry page of SAP Build Apps.

  1. In the Lobby, use the search field to look for the project named TA – App UI Template.

  1. Once you have found the template app, click the Options button and select Save As New Project.

  1. On the next screen, enter the following:

    Input Field Input Value
    Project Name Create Sales Order AC096819U21
    Description SAP Build Apps Sales Order Project AC096819U21
    • Click Save As New.

Once your project is saved it should open the SAP Build Apps editor automatically, as below. If the project does not open automatically, simply click your project’s name in the lobby.
