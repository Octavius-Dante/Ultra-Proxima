Bind page variables to UI elements

Bind page variable to UI elements

We created page variables for the sales order data, but now we need to get the data entered by the user into the respective variable. We do this by binding the each page variable to their respective UI elements, specifically, to the input boxes.

Whenever someone types into the input box, the value is automatically copied into the corresponding page variable.

  1. Go back to View so you can see the UI canvas.

  2. Using the TREE structure, click on the first input field (for Customer).

    In the Properties tab, click the X next to the Value field

    Select Data and Variables > Page variable

    then select customer.

    Click Save

  3. Using the TREE structure, click on the second input field (for Material).

    In the Properties tab, click the X next to the Value field, and select Data and Variables > Page Variable > material.

    Click Save

  4. Using the TREE structure, click on the third input field (for Quantity).

    In the Properties tab, click the X next to the Value field, and select Data and Variables > Page Variable > quantity.

    Click Save

  5. Using the TREE structure, click on the fourth input field (for Delivery Date).

    In the Properties tab, click the X next to the Value field, and select Data and Variables > Page Variable > date.

    Click Save

  6. Click Save (upper right).
