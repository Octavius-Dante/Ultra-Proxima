Overview

Overview of Sales order Process for SAP Work Zone

Create a Workspace with web content, a forum and a feed. You will then enrich it by integrating an application that you created with SAP Build Apps, which lets you trigger a process in SAP Build Process Automation. Later, you will develop and integrate your own UI integration card that surfaces data from a backend system. After completing this group, you will have your own engaging Workspace and learn how to create your own digital workplace.

Note: Make sure to exactly follow the exercise documentation and use the provided naming conventions. Otherwise, you will encounter issues.

Prerequisites

  • Your AC096819U21 user and Password
  • You have access to the SAP Build Work Zone Lobby
  • You have created an app in SAP Build Apps and a process in SAP Build Process Automation or use our workshop template process and template app

Goal

You will learn

  • How to create a workspace and add content to it that you can share with other members of your workspace.
